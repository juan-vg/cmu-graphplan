# Graphplan
Carnegie Mellon University Graphplan Impl

## Compile (on Linux)

    ~/cmu-graphplan$ cd source
    ~/cmu-graphplan/source$ make

## Exec example (on Linux)

    ~/cmu-graphplan$ source/graphplan -o tasks_ops.txt -f tasks_facts.txt -d
